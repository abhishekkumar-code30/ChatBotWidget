# Universal Chat Widget

A lightweight, customizable, and framework-independent chat widget for websites.

It enables businesses to provide instant customer support through predefined conversation flows without requiring AI or a backend. Users can browse FAQs, products, pricing, contact information, order support, and other business-specific content using an intuitive menu-driven interface.

## Features

- Lightweight and fast
- Pure HTML, CSS & JavaScript
- No dependencies
- Responsive design
- Customizable appearance
- Menu-based conversations
- Multi-level navigation
- Session memory
- Easy integration into any website

## Use Cases

Suitable for:

- Business websites
- E-commerce stores
- Portfolios
- Agencies
- Restaurants
- Educational institutions
- Healthcare websites
- Service providers
